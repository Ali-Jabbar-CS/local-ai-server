# Local AI Server

This project demonstrates a locally hosted AI environment capable of running Large Language Models (LLMs) and Stable Diffusion image generation models without external APIs.

## Tech Stack

- Python
- Linux
- WSL
- ComfyUI
- Stable Diffusion

## Features

- Local AI inference environment
- Image generation pipelines using ComfyUI
- Prompt experimentation and workflow tuning
- Image upscaling and quality improvements
